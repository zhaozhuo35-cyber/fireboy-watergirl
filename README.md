# 森林冰火人 - 双人闯关游戏

一个经典的双人协作平台游戏，灵感来源于4399小游戏。

## 在线游玩

游戏已部署到GitHub Pages：`https://你的用户名.github.io/fireboy-watergirl/`

（部署后将更新链接）

## 游戏特色

- 🎮 双人协作游戏模式
- 🔥 火人：可以穿过岩浆，使用方向键控制
- ❄️ 冰人：可以穿过冰水，使用WASD控制
- 💚 绿色毒液两人都不能碰触
- 💎 收集宝石后到达门口即可过关
- 🗺️ 5个精心设计的关卡，难度递增
- ✨ 精美的视觉效果和动画
- ⭐ 粒子特效系统

## 游戏操作

**火人**（红色）：
- ⬅️ 左移：左方向键
- ➡️ 右移：右方向键
- ⬆️ 跳跃：上方向键

**冰人**（蓝色）：
- ⬅️ 左移：A键
- ➡️ 右移：D键
- ⬆️ 跳跃：W键

**其他操作**：
- Enter：进入下一关
- R：重新开始当前关卡

## 关卡设计

1. **森林入口** - 简单教学关，熟悉基本操作
2. **岩浆洞穴** - 引入危险区域和跳跃机制
3. **天空之桥** - 需要精准的跳跃和协作
4. **毒液迷宫** - 复杂路径，考验协作能力
5. **终极挑战** - 综合考验所有技巧

## 本地运行

1. 下载所有文件到本地
2. 直接用浏览器打开 `fireboy-watergirl.html`
3. 开始游戏！

## 技术栈

- HTML5 Canvas
- CSS3
- JavaScript (ES6+)
- 无需任何外部依赖

## 游戏规则

- 火人碰到水会死亡
- 冰人碰到岩浆会死亡
- 两人碰到绿色毒液都会死亡
- 必须收集对应颜色的宝石才能开门
- 两人都到达各自的门才能过关

## 开发说明

使用原生JavaScript开发，包含以下系统：
- 物理引擎（重力、碰撞检测）
- 粒子特效系统
- 关卡管理系统
- 动画系统

## 部署到GitHub Pages

1. 在GitHub上创建新仓库（例如：fireboy-watergirl）
2. 将代码推送到仓库
3. 在仓库设置中启用GitHub Pages
4. 选择main分支作为源
5. 访问 `https://你的用户名.github.io/仓库名/fireboy-watergirl.html`
